# Skills Recomendadas para BGR-SHRIMP

Skills de [skills.sh](https://skills.sh/) seleccionadas para el proyecto BGR-SHRIMP (WhatsApp chatbot con FastAPI, OpenAI, Twilio, PDF generation, Google Sheets).

---

## Altamente Recomendadas

### systematic-debugging
- **Instalacion:** `npx skillsadd systematic-debugging`
- **Descripcion:** Patrones de debugging sistematico y metodico
- **Uso en el proyecto:** Depurar problemas en el bot de WhatsApp, pipeline de datos y flujos de conversacion

### test-driven-development
- **Instalacion:** `npx skillsadd test-driven-development`
- **Descripcion:** Metodologia TDD para desarrollo guiado por tests
- **Uso en el proyecto:** Mantener y mejorar la cobertura de tests (80%+ minimo) en los 30+ archivos de test existentes

### webapp-testing
- **Instalacion:** `npx skillsadd webapp-testing`
- **Descripcion:** Mejores practicas de QA para aplicaciones web
- **Uso en el proyecto:** Testing de la API FastAPI, endpoints de webhook y dashboard de review

### pdf
- **Instalacion:** `npx skillsadd pdf`
- **Descripcion:** Manejo y manipulacion de archivos PDF
- **Uso en el proyecto:** Mejorar la generacion de PDFs de cotizaciones con ReportLab

### seo-audit
- **Instalacion:** `npx skillsadd seo-audit`
- **Descripcion:** Auditoria y optimizacion SEO
- **Uso en el proyecto:** Optimizar landing page o documentacion publica del servicio BGR Export

## Recomendadas para el Stack

### database-schema-design
- **Instalacion:** `npx skillsadd database-schema-design`
- **Descripcion:** Modelado y diseno de esquemas de bases de datos
- **Uso en el proyecto:** Evolucionar el esquema SQLite/PostgreSQL de training capture y sesiones

### deployment-automation
- **Instalacion:** `npx skillsadd deployment-automation`
- **Descripcion:** Automatizacion de releases y despliegues
- **Uso en el proyecto:** Complementar el CI/CD con GitHub Actions para deploy a Render/Railway

### xlsx
- **Instalacion:** `npx skillsadd xlsx`
- **Descripcion:** Manejo avanzado de archivos Excel/spreadsheets
- **Uso en el proyecto:** Mejorar el manejo de Excel como fallback de datos de precios de camaron

### firecrawl
- **Instalacion:** `npx skillsadd firecrawl`
- **Descripcion:** Web scraping y extraccion de datos
- **Uso en el proyecto:** Enriquecer el sistema RAG o datos de precios con informacion de la web

### copywriting
- **Instalacion:** `npx skillsadd copywriting`
- **Descripcion:** Creacion y optimizacion de contenido escrito
- **Uso en el proyecto:** Mejorar mensajes del bot, templates de WhatsApp y contenido de PDFs de cotizacion
